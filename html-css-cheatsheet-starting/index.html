    HTML rules

    Here, I desrcibed all basic rules that you will need, in order to build the structure of a website. These tags need to be nested in the same order as they are displayed hereunder.

    HTML tags What it is used for
    <!DOCTYPE html> This tag is used to indicate the browser that the document it is about to read, is a document that uses HTML language. It is always the first line of the code in the HTML file.
    <html> The <html> tag is used right after the <!DOCTYPE html> tag and indicates that, anything that is in between, can be interpreted by the browser as HTML code.
    <head> This tag gives more information to the browser about the web page. The <head> element contains the metadata for a web page. This element will also contain the title and different link tags.
    <body> The <body> tag will contain all the content that will be displayed on the website.
    <header> This tag is used to place all content that will be dispalyed at the top of the web page. Ex: banner, <h1> tag, ...
    <main> The <main> tag contains all the main content of the web page. It can be compared to the actual text of a book, after the cover page and the table of contents.
    <footer> Finally, the <footer> will contain all the elements that will be displayed in the footer of the web page.

    Those HTML tags can all contain multiple children tags that will allow you to give your own structure to your web page.

    CSS rules

    After describing some basic structural HTML rules, I am going to cite some basic CSS selectors that you can use to style your web page.

    CSS selectors What it is used for
    * This is the universal selector. It is used to apply the same style rules to all elements of any type. For instance: apply the same font to all text on the page.
    p The type selector is used to target a specific type of elements. The CSS rule will then apply the same style to all elements of that type.
    .name The class selector is used to select an HTML element by its class. Here, for instance, the .name selector will apply the style to elements as follows: <p class='name'>.
    #name Here, the id selector will target one single element. Id's can have only one value and can be used only once per page. For instance, the #name selector will target the <h1 id='name'>.
    [href] The attribute selector is used to target HTML elements that already contain attributes. Here all elements that contain a [href] attribute will be styled.
    p : hover The pseudo-class selector can give a certain style to an element after a specific interaction with that element has been done. For instance, here, the style will be given to the p element when the mouse of the user hovers over it.
    h1.name Finally, chaining can be used to target a certain class nested in a parent element. Here, for example, the .name class inside of the <h1> element will be styled.

    All those selectors can contain mutliple rules that will give your web page its unique style.
